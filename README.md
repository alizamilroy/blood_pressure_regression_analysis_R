# Blood Pressure Regression Analysis
An analysis into the variables affecting systolic blood pressure in ICU patients in the New York Downtown Hospital.

This project investigates a possible relationship between systolic blood pressure, and whether it can be predicted based off   
a patients age and infection status. The data is first explored to investigate any possible relationships. Then, 
linear regression models are fitted to find the best-fitting model. The report was written using 
Quarto and is available in the repository.
